# Python NumPy OOP Mini Projects


## Overview

Implemented core Python programming concepts including string manipulation, file handling, object-oriented programming, and NumPy-based data processing.

## Tasks

* String operations: full name generation and alternate character extraction
* File processing: word count from input file and output storage
* Data transformation: height conversion using loops and list comprehensions
* OOP: Employee class with inheritance and average salary computation
* NumPy: vector creation, reshaping, and row-wise max replacement using vectorization

## Tech Stack

* Python 3
* NumPy

## Run

```bash
python src/word_count.py
python src/numpy_operations.py
```
